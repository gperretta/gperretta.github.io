Project details

Project goal

Rethinking and redesigning an app following the Apple's HIG was the main theme and goal of a solo-project at the Apple Developer Academy.
I've decided to use the UnicoCampania (iOS) App, which is an app I use daily and know quite well.
Personal goal

As a personal goal, I've decided to design an app that was able to give you access to your tickets in just one tap. I wanted an easy-to-use and clean-designed app with accessible colors and a clear user flow.

Development and Technologies

After a couple of lo-fi prototypes and multiple hand-drawn sketches, I've created the first hi-fi protoype of the app, by using Figma as an editing and prototyping software.
I've decided to implement the front-end side of the Tickets tab and that's when the Figma prototype became an app, running on an iPhone, too.
Here I had the chance to explore Swift and SwiftUI using native component.
I've used a horizontal scrollview for the "My tickets" section and a list of cards in the "Buy a Ticket" section.
Then I've also implemented a full-screen modal that can be openend just by tapping on a bought ticket and that will display all the ticket informations, its state and a QR code.
The QR code is generated from the user mail and a random alphanumeric sequence to simulate it uniqueness, by using CoreImage framework. Its functionality is also enhanced by the use of a timer: the code can be refreshed both manually, by tapping "Refresh", or automatically when the timer expires.
